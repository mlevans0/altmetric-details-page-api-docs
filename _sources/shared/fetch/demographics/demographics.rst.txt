Altmetric categorizes users from some sources based on their posting history and profile information. Counts for each category are included in this section along with geo-location data.

.. list-table:: 
   :widths: 10 10 80
   :header-rows: 1

   * - Key
     - Type
     - Description
   * - ``poster_types``
     - object
     - Groups of users who've mentioned the publication on Twitter. See :ref:`Poster types` for more information.
   * - ``users``
     - object
     - See :ref:`Users` for more information.
   * - ``geo``
     - object
     - Contains demographic mention information for Twitter, Facebook, News, and Policy. See :ref:`Geo` for more information.
